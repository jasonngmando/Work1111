# Monday Review Study Site

This is a small offline website (no server needed) to study the Monday Review PDF content.

## How to run
Option A (easiest):
- Double-click `index.html` to open it in your browser.

Option B (recommended if your browser blocks local scripts):
- Use a simple local server:
  - If you have Python:
    - Open a terminal in this folder
    - Run: `python -m http.server 8000`
    - Open: http://localhost:8000

## Features
- Flashcards (flip + spaced repetition style scheduling)
- Quiz (multiple choice or type answer)
- Browse (searchable Q/A list + cheat sheet)
- Progress page (export/import your progress)

Your progress is stored locally in your browser (localStorage).
